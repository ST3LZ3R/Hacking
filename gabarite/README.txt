#### Gabarite (https://www.gabarite.com.br/)
+--+ Sistema operacional utilizado +--+
Debian 11

+--+ Bibliotecas necessárias +--+
bash, curl

+--+ Execução+--+
./gabarite.sh ID_QUESTAO ALTERNATIVA 

ID_QUESTAO: Código ID da questão, normalmente no começo da questão (Ex: '208935' é o id dessa https://www.gabarite.com.br/questoes-de-concursos/208935-questao).
ALTERNATIVA: Alternativa escolhida para teste em minúsculo (Ex: d).

** Vídeo Explicativo (tutorial.mkv) - Acaso erre a alternativa, é retornado a alternativa correta **
** Lembrar de adicionar permissão de execução com 'chmod +x' ao script .sh **
